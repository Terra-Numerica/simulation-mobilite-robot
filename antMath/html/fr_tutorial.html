Bienvenue dans le tutoriel de l'application web "Fourmilière Animée" !

Cette application vous permet de dessiner un chemin sur le terrain de jeu, suivi par des fourmis en file indienne. Voici comment l'utiliser :

  1. Dessin du chemin : Utilisez votre doigt ou la souris pour dessiner le chemin que les fourmis suivront.
  2. Fourmilière et Fourmis : Les fourmis partent de la fourmilière au début du chemin. La première fourmi suit le tracé, les autres la suivent en file indienne.
  3. Changer le fond d'écran : Choisissez parmi les fonds proposés ou personnalisez le vôtre.
  4. Changer la langue : Sélectionnez votre langue préférée.
  5. Téléchargement de GIF : Une fois le chemin tracé, téléchargez un GIF de l'animation des fourmis.
  6. Section "Contrôle" : Personnalisez le comportement des fourmis :
    • Changez la vitesse d'animation.
    • Modifiez l'espacement entre les fourmis.
    • Sélectionnez un autre chemin dessiné.
    • Arrêtez ou reprenez l'animation.
    • Réinitialisez le jeu.
    • Affichez ou masquez tous les chemins dessinés.
  7. Courbe de la taille du chemin : Visualisez la courbe de la taille du chemin en fonction du nombre de fourmis.
  8. Section "Documentation" : Informations détaillées sur les algorithmes utilisés.

Amusez-vous bien avec "Fourmilière Animée" ! N'hésitez pas à explorer toutes les fonctionnalités et à personnaliser votre expérience. Des questions ? Consultez la section "Documentation". Bon divertissement !
