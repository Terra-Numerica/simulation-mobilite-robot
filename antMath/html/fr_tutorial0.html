Tutoriel

La simulation se déroule sur le terrain d'herbe.
Pour commencer, il faut dessiner une ligne qui représentera le chemin pris par la première fourmi.
Pour cela, il suffit de dessiner le chemin souhaité avec la souris en maintenant le clic gauche enfoncé. Relâcher le clic gauche placera le point de départ et d'arrivée pour toutes les fourmis.
Vous pouvez modifier à votre guise les paramètres de la simulation dans le panneau de contrôle.
Vous pouvez également modifier la langue de la simulation en cliquant sur le bouton en haut à droite.
Le graphique illustre en temps réel le nombre de pas effectués par chaque fourmi depuis leur point de départ jusqu'à leur point d'arrivée.
